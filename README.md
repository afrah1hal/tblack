:: b.lack
=========

Black is a library that implements standard JavaScript functions, which
are buggy or lack support in older browsers, in pure JavaScript.


Running tests
-------------

Unless you first write them yourself or wait until I'm not so busy, no
can do sweetie...


Documentation
-------------

Refer to the source. And the MDC/MSDN pages.


Reporting issues/patches
------------------------

Use the Github tracker. Remember to follow the library's
conventions. Take a look at Bone's README file for a simple list of
those.
